# Screening Shariah AAOIFI — Bourse de Casablanca

Outil de screening de conformité islamique pour les sociétés cotées au MASI, basé sur le Standard Shariah N°21 de l'AAOIFI.

## Méthodologie

Le screening se fait en deux étapes :

**1. Screening qualitatif** — exclusion des secteurs haram : banques conventionnelles, assurances conventionnelles, alcool, tabac, jeux de hasard.

**2. Screening quantitatif** — trois ratios calculés sur la capitalisation boursière :

| Ratio | Seuil |
|---|---|
| Dettes à intérêts / Capitalisation | ≤ 30% |
| Dépôts rémunérés / Capitalisation | ≤ 30% |
| Revenus non-halal / Revenus totaux | ≤ 5% |

 

## Données

Les données financières sont disponibles dans les rapports annuels publiés sur le site de l'[AMMC](https://www.ammc.ma) et de la [Bourse de Casablanca](https://www.casablanca-bourse.com).

## Avertissement

Outil fourni à titre indicatif. Ne constitue pas un avis Shariah certifié.
